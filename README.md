# Vpn
Стандартный VPN Manager 

Функции:

1 Подключение к разным серверам по протоколу Ipsec/Ikev2

2 Бесплатная версия на 7 дней

3 Покупка преимиум аккаунта дающего безграничный доступ к ВПН
